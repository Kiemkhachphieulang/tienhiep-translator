🧠 Cerebras API Key (Ưu tiên 2 · 70K token/phút) ✗ Chưa có
Cách lấy Cerebras API Key miễn phí:
1. Vào cloud.cerebras.ai → Sign up
2. Vào API Keys → Create new key
3. Copy key → dán vào đây → Test → Lưu
✓ GPT-OSS 120B · backup khi Gemini rate limit
🧠 Miễn phí · 70,000 token/phút
⚡ Groq API Key (Ưu tiên 3 · 6K token/phút) ✗ Chưa cấu hình
Cách lấy Groq API Key miễn phí:
1. Mở Chrome → console.groq.com
2. Đăng ký / đăng nhập (dùng Gmail)
3. Vào API Keys → Create API Key
4. Copy key (bắt đầu bằng gsk_...) → dán vào đây
⚡ Miễn phí 14,400 req/ngày · 6,000 token/phút
